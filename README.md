# RySes Virtual App
> Is an application developed and designed as an encryption app.

- [x] String to Ascii

- [ ] Ascii to String
  
- [x] String to Binary

- [x] Binary to String

- [x] String to Hex

- [x] Hex to String

- [x] String to Base64

- [x] Base64 to String

- [x] String to Morse Code

- [ ] Morse Code to String
  
- [x] String to SHA-1

- [x] String to SHA-256

- [x] String to MD5


> With self customized android.widget.*
